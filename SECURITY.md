# Reporting Security Issues

If you need to report a security issue in this documentation project or the
associated AISVS content, contact the project leaders listed on the project page:

- [Jim Manico](mailto:jim@owasp.org)
- [Russ Memisyazici](mailto:russ.m@owasp.org)

Please avoid filing public issues for sensitive reports. Share enough detail to
reproduce the problem, understand impact, and validate a fix.
